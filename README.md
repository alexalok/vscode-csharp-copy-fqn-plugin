# C# Copy Reference

VS Code extension command for copying fully qualified C# type and member names.

Command: `csharp-copy-reference.copyFullyQualifiedName`

Suggested keybinding:

```json
{
  "key": "cmd+alt+c",
  "command": "csharp-copy-reference.copyFullyQualifiedName",
  "when": "editorTextFocus && editorLangId == csharp"
}
```

Windows/Linux:

```json
{
  "key": "ctrl+alt+c",
  "command": "csharp-copy-reference.copyFullyQualifiedName",
  "when": "editorTextFocus && editorLangId == csharp"
}
```
